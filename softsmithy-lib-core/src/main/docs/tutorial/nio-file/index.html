NIO File

The SoftSmithy Utility Library provides various utility classes for the NIO File API introduced in Java SE 7.

Copy files

Below is the code from CopyFilesSample.java that copies some files to a target directory.
This sample uses the CopyFileVisitor to recursively copy a source directory to a target directory. The CopyFileVisitor implements the FileVisitor interface and thus can be use with one of the Files.walkFileTree methods. For convenience there is the static CopyFileVisitor.copy method. To run this sample, use: 
        cd docs/samples/dist
        java -cp softsmithy-lib-samples-${project.version}.jar:lib/softsmithy-lib-core-${project.version}.jar samples.nio.file.CopyFilesSample src/main/data/first output

Extract resources from a JAR

Below is the code from ExtractJarResourceSample.java that extracts a resource from the JAR containing the ExtractJarResourceSample.class file to a target directory.
Oracle's Java SE 7 Runtime comes with a FileSystemProvider implementation for JAR and ZIP files. This FileSystemProvider allows you to work with JAR and ZIP files similar to the way you can work with the local file system.

JarFiles provides some utility methods to work with JAR/ ZIP files. With the help of CopyFileVisitor you can easily recursively extract/ add directories from/ to JAR or ZIP files.
To run this sample, use: 
        cd docs/samples/dist
        java -cp softsmithy-lib-samples-${project.version}.jar:lib/softsmithy-lib-core-${project.version}.jar samples.nio.file.ExtractJarResourceSample output
